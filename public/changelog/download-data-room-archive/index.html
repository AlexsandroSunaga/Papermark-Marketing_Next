Download Entire Data Room as Archive

Download complete data rooms as zip files with one click

Need an offline backup or local copy of your data room? You can now download the entire data room as a zip archive.

Key features:

  • One-click download — get all folders and files in a single zip file
  • Preserves structure — folder hierarchy maintained in the archive
  • Large data room support — handles data rooms with hundreds of documents
  • Owner-only access — available to data room owners and admins

Perfect for creating backups before major changes, archiving completed deals, or preparing offline review copies for stakeholders. Find the download option in your data room settings.
